Planned support
---------------
* First Generation RX-8


Building
--------
### Requirements
* Qt5
* CMake 2.8.11 or higher

### Build instructions

`git clone https://github.com/Libretuner/LibreTuner.git`

`git submodule update --init --recursive`

`cd LibreTuner`

`cmake .`

`make`

`./LibreTuner`
